# ssm-edit

Edit a parameter  on Systems Manager parameter store.

Inspiread by https://github.com/tsub/s3-edit


# Install

Download from release page.


# Usage

```
$ ssm-edit list
$ ssm-edit edit
$ ssm-edit version
```
